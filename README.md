# 国の情報アプリ

## 概要
このアプリは、ユーザーが国の名前を入力すると、その国についての情報を自動で調べて表示してくれるものです。具体的には、首都や人口、どの地域にあるかといった基本情報に加えて、国旗の画像も見ることができます。

## 使用したAPI
・REST Countries API
